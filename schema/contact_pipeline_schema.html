Google Antigravity — Contact Pipeline Architecture
Google Antigravity — Contact Form Pipeline  |  POST /webhook/contact
HTTP Webhook  /  Trigger POST /webhook/contact  (main.py)
Accepts JSON  { name, email, message }  · HMAC-SHA256 verification
400 Invalid JSON  ·  401 Bad signature  ·  422 Missing fields
↓
process_contact()  —  Step 1  —  Validate & Process name  →  strip   |   email  →  lowercase + regex validate   |   message  →  strip
contact_id: UUID[:8].upper()  ·  created_at: UTC ISO-8601  ·  status: "New"
↓
append_contact_to_sheet()  — Step 2 Google Sheets API v4  /  OAuth2
Tab: Contacts  (auto-created)
Cols: id · date · name · email · msg · status
Fallback: data/contacts.csv Google Sheets CRM
categorise_and_summarise()  — Step 3 Gemini 2.0 Flash Lite
category  →  domain label
priority  →  HOT / WARM / COLD
summary  →  ≤ 30-word brief AI Completion
↓
JSON Response  —  HTTP 200 { status, contact_id, name, email, message, crm_row, ai_category, ai_priority, ai_summary }
